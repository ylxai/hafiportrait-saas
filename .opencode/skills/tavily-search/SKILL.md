# Tavily Search

## What It Does
Web search for research, finding solutions, and getting current information.

## When to Use
- Researching solutions
- Finding best practices
- Getting current information
- Looking up error messages
- Technical research

## How to Use
Add "use tavily" to your prompt:

```
How to fix Next.js hydration error? use tavily
```

```
What are the best practices for React 19? use tavily
```

## Tips
- Be specific with search query
- Combine with sequential thinking for complex research
- Use for technical issues not covered by context7
